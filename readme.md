# Xpath QS Web

Framework for the interaction with Web-Based drivers

## Usage

Apache Maven

```xml

<dependency>
    <groupId>org.xpathqs</groupId>
    <artifactId>xpathqs-web</artifactId>
    <version>0.0.1</version>
</dependency>
```

Gradle Kotlin DSL

```kotlin
implementation("org.xpathqs:xpathqs-web:0.0.1")
```

Gradle Groovy DSL

```groovy
implementation 'org.xpathqs:xpathqs-web:0.0.1'
```
