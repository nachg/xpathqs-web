#Xpath QS Driver

Framework for the interaction with drivers for the xpathqs-core


## Usage

Apache Maven
```xml
<dependency>
  <groupId>org.xpathqs</groupId>
  <artifactId>xpathqs-driver</artifactId>
  <version>0.0.2</version>
</dependency>
```

Gradle Kotlin DSL
```kotlin
implementation("org.xpathqs:xpathqs-driver:0.0.2")
```

Gradle Groovy DSL
```groovy
implementation 'org.xpathqs:xpathqs-driver:0.0.2'
```
